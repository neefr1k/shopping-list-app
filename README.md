##  Описание проекта

**Сервис управления списком покупок** — это веб-приложение, которое помогает пользователям:
- добавлять товары в список покупок
- просматривать актуальный список
- удалять выполненные позиции

Приложение имеет минималистичный интерфейс и не требует регистрации.

##  Требования к системе

### Функциональные требования
| ID | Требование | Описание |
|----|------------|-----------|
| FR1 | Добавление товара | Пользователь может добавить новый товар через текстовое поле |
| FR2 | Просмотр списка | Все добавленные товары отображаются в виде списка |
| FR3 | Удаление товара | Пользователь может удалить любой товар из списка |
| FR4 | Подтверждение удаления | При удалении появляется диалог подтверждения |

### Нефункциональные требования
| ID | Требование | Описание |

##  Макеты интерфейса
+------------------------------------------+
|  Список покупок |
+------------------------------------------+
| [ Напр: молоко, хлеб... ] [➕ Добавить] |
+------------------------------------------+
| • молоко [✖ Удалить] |
| • хлеб [✖ Удалить] |
| • яблоки [✖ Удалить] |
| |
| (если список пуст: " Список пуст") |
+------------------------------------------+

### Добавление товара
1. Пользователь вводит текст в поле
2. Нажимает кнопку "Добавить"
3. Товар появляется внизу списка

### Удаление товара
1. Пользователь нажимает "[✖ Удалить]" рядом с товаром
2. Появляется подтверждение: "Удалить молоко?"
3. Товар исчезает из списка

##  Описание API

Приложение предоставляет следующие endpoints:

| Метод | Путь | Описание | Параметры |
|-------|------|----------|-----------|
| GET | `/` | Главная страница | - |
| POST | `/add` | Добавить товар | `item` (form data) |
| GET | `/remove/<item>` | Удалить товар | `item` (в URL) |

### Примеры запросов

**Добавление товара:**
POST /add
Content-Type: application/x-www-form-urlencoded

item=молоко
GET /remove/молоко

Модель данных
В текущей версии используется упрощённое хранение в памяти сервера.

Структура данных
python
shopping_items = []  # список строк (list of str)
Описание полей
Поле	Тип	Описание	Пример
item	string	Название товара	"молоко"
Ограничения
Длина товара: 1-100 символов

Регистр сохраняется как введён

Данные не сохраняются после перезапуска сервера
