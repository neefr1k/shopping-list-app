# Список покупок

Веб-приложение для управления списком покупок

## Ссылки

- **Сайт:** https://neefr1k.pythonanywhere.com
- **Репозиторий:** https://github.com/neefr1k/shopping-list-app

## Технологии

- Python + Flask
- HTML + CSS
- PythonAnywhere

## Функции

- Добавление товара
- Просмотр списка
- Удаление товара

## Структура
app.py # главный файл
database.py # работа со списком
templates/ # HTML шаблоны


## Запуск
pip install flask
python app.py

## Вывод
Разработано веб-приложение, соответствующее требованиям ТЗ. Все функции работают. Сайт опубликован.
